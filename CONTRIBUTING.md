# Contributing

Thank you for contributing to the Campus Lost & Found Portal.

## Steps

1. Clone the repository.
2. Create a new branch.
3. Make your changes.
4. Commit your work.
5. Push the branch.
6. Create a Pull Request.

## Code Style

1.Use consistent indentation (4 spaces).
2.Write clear commit messages.
3.Keep functions modular and documented.

## Documentation

1.Update `README.md` and `CHANGELOG.md` when adding new features.
2.Add comments in code for clarity.

## Community

1.Be respectful and collaborative.
2.No spam or irrelevant PRs.
3.Follow the project license.
