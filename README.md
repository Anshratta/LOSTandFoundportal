# Campus Lost & Found Portal

A web application that helps students report and recover lost items on campus.

---

## Features

- Report Lost Item
- Report Found Item
- Browse Reported Items
- Search Items
- Filter Lost & Found Items
- Firebase Firestore Integration
- Responsive Design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Firebase Firestore
- Git & GitHub
- Vercel
