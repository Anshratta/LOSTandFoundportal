# Campus Lost & Found Portal

A responsive web application that helps students report, browse, search, and recover lost and found items within a university campus.

## Live Demo

Lost and Found Portal:-> https://lost-and-foundportal.vercel.app/ 

---

## Project Overview

The Campus Lost & Found Portal provides a centralized platform where students can report lost belongings, submit found items, and browse existing reports. The project aims to simplify the recovery process by maintaining a cloud database of reported items using Firebase Firestore.

---

## Features

- Report Lost Item
- Report Found Item
- Browse Reported Items
- Search Items
- Filter Lost & Found Items
- Firebase Firestore Integration
- Responsive Design

## 🔥 Firebase Integration

The project uses Firebase Firestore as the cloud database.

Firebase is responsible for:

- Storing Lost Item Reports
- Storing Found Item Reports
- Retrieving Reported Items
- Updating Item Information
- Providing Real-Time Cloud Storage

---

## Deployment

The application is deployed using *Vercel*, allowing users to access it directly through a web browser without any local installation.
